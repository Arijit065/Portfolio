# Arijit Mandal — Developer Portfolio

A high-performance, responsive developer portfolio built with React & Vite, featuring a terminal/tech theme, smooth animations, and excellent mobile experience.

## 🚀 Features

- ⚡ **Lazy Loading** — Heavy components loaded on demand
- 🎯 **Service Worker** — Advanced caching strategies & offline support
- 📱 **Mobile-First** — Optimized for all screen sizes
- 🎨 **Smooth Animations** — GPU-accelerated with Framer Motion
- 🖥️ **Terminal Theme** — Classic terminal green/cyan aesthetic with matrix effects

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React 18 | UI Framework |
| Vite | Build Tool |
| Tailwind CSS | Utility-first Styling |
| Framer Motion | Animations |
| Redux Toolkit | State Management |
| EmailJS | Contact Form |

## 🚀 Quick Start

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Arijit065/Portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
npm run preview
```
